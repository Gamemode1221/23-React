## 4주차
### 2023.03.23 목요일
> ### 강의
> 리액트 JSX 특징  
> ##### 장점
> 1. 코드가 간결해짐.
> 2. 가독성이 향상됨.
> 3. Injection Attack이라 불리는 해킹 방법을 방어함으로 보안에 강함.
> ##### JSX 사용법
> 1. 모든 자바스크립트 문법을 지원함
> 2. 자바스크립트 문법에 XML과 HTML을 섞어서 사용함
> 3. 아래 코드의 2번 라인처럼 섞어서 사용하는 것
> 4. 만일 html이나 xml에 자바스크립트 코드를 사용하고 싶으로 중괄호를 사용함
 ```jsx
const name = "소플";
const element = <h1>Hello, {name}</h1>;
```
